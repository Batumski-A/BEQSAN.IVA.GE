# BEQSAN — Premium Window Manufacturer

Architectural-firm-meets-craftsman-atelier. Closer to David Chipperfield, Vitsoe, or an architecture monograph than to a SaaS landing page. Roman Sharashidze has hand-built aluminium + PVC windows in Salibauri, Batumi since 1998. Audience: wealthy private homeowners + boutique hospitality developers buying a 25-year product.

## Tone

Restrained. Editorial. Material-honest. Confident but never loud. Premium feels uncrowded, not busy.

## Color (dark, warm)

- bg: #14110F — deep graphite, warm-leaning, no blue cast
- bg-elevated: #1B1816 — surfaces sitting forward
- bg-raised: #231F1C — cards
- fg-primary: #F2EBE2 — warm white
- fg-secondary: #A8A097 — warm grey
- fg-tertiary: #6E665D — captions
- hairline: #F2EBE2 at 8% opacity
- accent: single restrained amber #B8943F — only on hero CTA, one section eyebrow per page, one pull-quote, active nav underline. Never more than one amber element per viewport.

Five neutrals + one amber. That restraint IS the premium signal.

## Typography

- Display headline: serif, high-contrast, upright (not italic). 72-120px, tight leading 0.95, heavy 700-900. Hero + section H1 only. Use Newsreader or Vollkorn.
- Body + UI: grotesque sans. Inter or Hanken Grotesk. 16px floor, 1.55 leading.
- Mono labels: JetBrains Mono. Drawing markers (W/H/U-value), numeric specs, section numbering like "№ 02".

Serif display + grotesque sans + technical mono — the trio behind Chipperfield, Snøhetta, Foster + Partners.

## Layout

- 12-col grid, 1440px max, but content uses 5-7-col asymmetry with deliberate negative space.
- Generous section padding: 128-180px vertical desktop, 96-128px mobile. Premium feels uncrowded.
- Hairline rules everywhere — 1px @ 8% opacity. Never thick.
- Drawing markers — mono-spec callouts (W: 120 სმ, H: 145 სმ, U: 1,2 W/m²K) in a vertical column adjacent to the hero. Editorial signature.
- Hero pages are magazine layouts (long-scroll editorial with image plates, pull-quotes, pace shifts). Cards only in catalog/gallery.

## Imagery

No real photos yet. Hero plates: full-bleed wide SVG architectural drawings, dark warm tone, single amber-tinted highlight. Treat illustrations like an architecture monograph cover.

## Motion

Subtle vertical lift on scroll-in (16px → 0, opacity 0.6 → 1, ~700ms). No bounce, no slide-from-edge, no flip cards. Respect prefers-reduced-motion.

## Anti-checklist (do NOT generate)

- Purple/blue gradients on hero
- Rounded "feature card" grids with icons
- Stock photo of smiling office worker
- More than one amber element per viewport
- Round avatars next to testimonials
- "Trusted by" logo strip
- Numbered animated counters
- Friendly emoji in headings
- Italic display serif
- Drop shadows behind cards (use hairline rules)
- Three-column equal-width "Features" sections
